# 📊 Insights from Data Analysis

## 🔍 Key Observations

- ~9% users are at churn risk
- Negative reviews strongly correlate with low ratings (1–2 stars)
- Major churn reasons:
  - Driver cancellations
  - Long wait times
  - App crashes
  - High pricing

## 📈 ML Model Results

- Best Model: Linear SVM
- Accuracy: ~92%
- F1 Score: ~76%

## 🧠 Feature Insights

Top churn indicators:
- "not"
- "cancel"
- "wait"
- "problem"
- "late"

## 📌 Platform Insights

- Uber: Large dataset, strong patterns
- Ola: Operational issues (cancellations, delays)
- SubRide (Rapido proxy): Driver behavior issues
