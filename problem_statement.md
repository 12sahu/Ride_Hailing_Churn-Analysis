# 🚨 Problem Statement

Ride-hailing platforms like Uber, Ola, and Rapido face high customer churn due to:

- Driver cancellations
- Long wait times
- High pricing (surge pricing)
- Poor app performance
- Lack of customer support

## 🎯 Objective

The goal of this project is to:

- Predict customer churn using machine learning
- Analyze key reasons behind churn
- Provide actionable business insights
- Simulate a real-world multi-agent AI system
